# WeatherApp
## 모바일에서 사용가능한 날씨 앱

<img width="400" height="800" alt="Image" src="https://github.com/user-attachments/assets/0b54c077-0eff-47e3-929e-de5bb6ff85d1" />

## 기술 스택

- Java
- OpenWeatherMap API



## 주요 기능

| 기능 | 설명 |
| ------ | ------ |
| 도시 선택 | 날씨를 확인하고 싶은 도시를 검색하여 선택 |
| 현재 날씨 | 현재의 온도, 날씨 확인 |
| 일주일 날씨 | 3시간 간격으로 온도, 날씨 확인 |


## 개발 환경

> SDK 34
> 
> IDE : AndroidStudio
